# Student_Management_System_BackendProject

# Description
This project is a backend REST API developed using Spring Boot.
It is used to manage student records by performing basic CRUD operations.
The application connects to a database to store and retrieve student information.
All APIs were tested using Postman.

**Technologies Used**
Java
Spring Boot
Spring Data JPA
Hibernate
MySQL
Maven

**Functionality**
Add student details
Update student information
Delete student records
View student by ID
View all students

**API Type**
RESTful Web Services

**Testing Tool**
Postman

**Note**
This is a backend-only project.
There is no user interface.
The APIs can be integrated with any frontend or client application.
